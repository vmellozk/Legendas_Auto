# Projeto de Automação de Download e Transcrição

## Visão Geral

Este projeto está sendo desenvolvido para automatizar o processo de download de áudio e vídeo do YouTube, transcrição do áudio em texto e download de vídeos do Instagram. Ele visa oferecer uma solução integrada para converter conteúdo de áudio em texto e baixar vídeos de redes sociais de maneira simplificada.

O projeto ainda está em fase de desenvolvimento e planeja incluir várias outras funcionalidades para ampliar suas capacidades.

## Funcionalidades Implementadas

- **Download de Áudio do YouTube:**
  - Baixa o áudio de vídeos do YouTube em formato MP3 utilizando yt_dlp.
  - Permite escolher o diretório de saída para o áudio baixado.

- **Transcrição de Áudio em Texto:**
  - Utiliza o modelo Whisper para transcrever o áudio baixado em texto.
  - Salva a transcrição em um arquivo de texto.

- **Download de Vídeos do Instagram:**
  - Baixa vídeos do Instagram usando Instaloader.
  - Permite escolher o diretório de saída para o vídeo baixado.

## Funcionalidades Planejadas

- **Conversão Automática de Vídeo em Áudio:**
  - Implementar a funcionalidade para converter vídeos do Instagram em áudio.

- **Interface Gráfica Melhorada:**
  - Aprimorar a interface usando PySimpleGUI para melhor experiência do usuário.

## Como Utilizar

1. **Pré-requisitos:**
   - Python 3.x instalado.
   - Instale as dependências listadas no arquivo `requirements.txt`.

2. **Configuração:**
   - Clone o repositório para sua máquina local.
   - Execute `pip install -r requirements.txt` para instalar as bibliotecas necessárias.

3. **Execução:**
   - Execute o arquivo principal do projeto.
   - A interface gráfica será aberta, permitindo que você cole URLs do YouTube e Instagram para iniciar os downloads e transcrições conforme desejado.

## Sobre o Projeto

### Objetivo

O projeto de automação de download e transcrição foi concebido para simplificar o processo de obter conteúdo de áudio e vídeo do YouTube e Instagram, além de facilitar a conversão de áudio em texto para diversos fins.

### Motivação

Com a crescente demanda por ferramentas que automatizem tarefas de mídia social e transcrição de conteúdo multimídia, este projeto visa oferecer uma solução acessível e eficiente para criadores de conteúdo e profissionais que trabalham com mídia digital.

### Funcionalidades Futuras

- **Suporte a Mais Plataformas:**
  - Integração com outras plataformas de mídia social para download de conteúdo.

- **Melhorias na Interface de Usuário:**
  - Aprimoramento da GUI para uma experiência mais intuitiva e amigável.

- **Expansão das Capacidades de Transcrição:**
  - Implementação de modelos de transcrição avançados para melhorar a precisão e o desempenho da transcrição de áudio.

Este projeto está comprometido em evoluir e adaptar-se às necessidades emergentes no campo de automação de mídia digital e transcrição de áudio.
